== Description ==

This plugin orgnizes the layout of categories boxes in the post add/edit screens and will work for all post types in your website. If you have a lot of categories in your posts and you face difficulties in creating posts because of a lot of scroll to select the appropriate category, then this plugin is for you.

I have updated the plugin and now the folding of categories will work on quick edit screen also.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Is there a setting to enable the folding option for categories? =

No, as long as you activate the plugin, it will work automatically.
